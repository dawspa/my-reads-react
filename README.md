# FEND Project - MyReads

Project for _Google Udacity Front End Nanodegree Scholarship_. Powered by `React`, `React-Router` and dynamic content rendering.

## Installation

To get started developing right away:

* install all project dependencies with `npm install`
* start the development server with `npm start`

### Specification

This website is an bookshelf management app emulation, which  lets the user query books and add them to his shelves, for current, previous and future reads.
